Descrição Detalhada

Classe Dbcommand gera todas as query SQL e já as executa, além de formatar o seu retorno da execução.

Também tem alguns outros métodos que são bem utéis.

Problema:
O metodo Select só gera query para condicionais AND, deixando de fora os demais condicionais (OR, NAND, NOR, etc).
